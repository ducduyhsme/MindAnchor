# MindAnchor

A cross-platform mental wellness app built with React Native (Expo).

## Features

- **Daily Activity Log** — rotating mindfulness prompts, emoji mood tracker, calendar history
- **AI Companion** — push-to-talk voice chat, TTS responses, crisis safety protocol
- **Community Forum** — anonymous threads, category filters, live audio rooms
- **Relaxation Library** — guided audio, box breathing animation, reading materials

## Getting Started

```bash
npm install
npx expo start
```

## Tech Stack

- React Native + Expo (TypeScript)
- expo-router for file-based navigation
- expo-av (audio), expo-speech (TTS)
- react-native-calendars
- @react-native-async-storage/async-storage
